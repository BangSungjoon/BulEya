# React + Vite

## 설치 옵션
- javascript
- React + SWC
- react-router-dom
- tailwind

## 프로젝트 구조
```
📁 src
 ┣ 📁 components        # 재사용 가능한 컴포넌트들
 ┣ 📁 pages             # 페이지 단위 컴포넌트 (라우팅용)
 ┣ 📁 context           # 전역 상태 관리 (ex. 다크 모드)
 ┣ 📁 hooks             # 커스텀 훅 (안쓰면 삭제 예정)
 ┣ 📁 styles            # CSS 또는 Tailwind 관련 스타일
 ┣ 📄 App.jsx           # 메인 컴포넌트
 ┣ 📄 main.jsx          # React 렌더링 진입점
 ┗ 📄 index.css         # 전역 스타일
```
